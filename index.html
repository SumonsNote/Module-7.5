HTML Inline Elements

Inline elements are used in html. it takes space only where they are used. No extra space as block elemnet does. It take side by side.

  2. button for button

  4. input for user input

HTML Block Elements

Block Elementes are used in HTML. It takes full space where they are used. It needs full structure in space and have alone.

  1. article for blog written
  2. div for container
  3. fieldset for selection
  4. footer for website content
  5. header for heading something
  6. main everything are here
  7. navigation is used top of a site
